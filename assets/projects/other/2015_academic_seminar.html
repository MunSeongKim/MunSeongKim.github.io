2015년 공과대학 학술제 순천향대학교 공과대학
프로젝트 이름 IoT 스마트 강의실
설명
강의실에 설치한 환경 센서 데이터를 수집하여 데이터베이스 서버에 저장
수집한 환경 데이터를 처리하여 상황인지 후 환풍기 등의 기기 제어 이벤트 발생
제 3자에게 강의실 환경 정보에 접근할 수 있는 방법 제공
기능
마이크로 프로세서(아두이노)와 임베디드 보드(비글본블랙)을 사용하여 수집 모듈 제작, 시리얼 통신을 기반으로 데이터 수집
릴레이 스위치를 사용하여 마이크로 프로세서에서 전자기기 제어 구현
JSP와 Jersey 프레임워크를 사용하여 RESTful API를 통해 실시간 환경 정보 제공
