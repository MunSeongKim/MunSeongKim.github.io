2017년 공과대학 학술제 순천향대학교 공과대학
프로젝트 이름 MVC 모델을 이용한 빅데이터 활용 웹 서비스 설계 및 구현
설명
국내여행에 관련된 블로그 포스트들을 수집하고 텍스트 분석을 통해 여행지 언급 횟수를 추출하여 지도 기반의 웹 서비스를 제공
수집: Python 언어를 기반으로 Selenium, PhanthomJS, BeautifulSoup4 모듈을 사용하여 포스트의 데이터 수집
처리: Elasticsearch의 Percolator 기능을 사용하여 여행지 추출 후 언급횟수 누적, 데이터 저장 및 검색 기능 구현
표현: Ruby on Rails 프레임워크를 사용하여 MVC 패턴을 적용하여 웹 서버 구현, jQuery와 구글지도 API로 인터페이스 구현
기능
구글 지도 마커를 통해 집계 결과 제공, 마커 클릭시 해당 지역의 시 단위로 확대
검색 기능 제공, 검색 시 해당 지역을 중심으로 옮기고 관련 블로그 10개 리스트를 좌측에 출력
RESTful API를 통해 지도의 확대 축소 시 마커 업데이트 실행
매시간 100개의 블로그 포스트를 수집하여 처리 후 저장
