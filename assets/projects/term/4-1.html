프로젝트 이름 Timing Shot
설명
중심에 고정되어 회전하고 있는 캐릭터를 클릭하여 탄환을 발사하여 별을 맞추는 게임
캐릭터를 누르고 있으면 회전하고 있는 속도가 점차 빨라짐
안드로이드 스튜디오와 내장 SQLite를 사용하여 게임 결과를 누적시키고 랭킹으로 보여줌
기능
캐릭터 회전, 탄환 발사, 랜덤한 위치에 타겟 생성 및 중심으로 진행하도록 설정
각도 변화량을 통해 타겟이 중심으로 이동하도록 구현
1개의 타겟이 제거되면 2개의 타겟이 생성
타겟을 제거한 점수를 누적하고 데이터베이스에 저장하여 사용자 랭크 출력

실행 영상
4학년 1학기 빅데이터이해 이상정 교수님
프로젝트 이름 MVC 모델을 이용한 빅데이터 활용 웹 서비스 설계 및 구현
설명
여행지역을 언급한 국내여행 블로그 포스트의 개수를 웹 기반 지도를 이용해 제공하는 웹 서비스 설계
국내여행 카테고리에 있는 네이버 블로그 포스트들을 수집
Elasticsearch를 이용하여 데이터 처리, 저장, 검색 기능 구현
Ruby 언어와 Rails 프레임워크를 사용하여 웹 서비스 구현
구글 지도 API를 사용하여 지도를 사용한 직관적 인터페이스 구현
기능
Ruby 기반의 웹 크롤러를 제작하여 포스트 수집
percolator를 사용한 여행 지역 언급횟수 누적
도 단위의 블로그 포스트의 개수 확인
지역명으로 검색하여 관련 블로그 포스트 목록 제공
